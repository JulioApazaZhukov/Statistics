# Coding style rules

### Placement of braces

```cpp
namespace example {
    //content
}
```

```cpp
void example ()         // or int or float or string
{
    //content
}
```

```cpp
int main ()
{
    //content
}
```

```cpp
if (condition) {
    //content
} else if (condition) {
    //content
}
```

```cpp
for (example) {           // or while
    //content
}
```

```cpp
do {
    //content
} while()
```
